# 📝 To-Do CLI App

A modular task management CLI application built with Python.

## Features
- Add, list, delete tasks
- Mark tasks as complete
- Persistent storage using SQLite
- Clean architecture (OOP + layered design)

## Run

```bash
python main.py add "Learn Python"
python main.py list.
